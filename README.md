# Sport Events Web App

## Authors

* Celina Ziane
* Said Amroun

## Project Description

This project is a web application that helps users find sport partners and discover local sport events.

Users can:

* Browse sport activities
* Create sport events
* Join events
* Create an account and log in
* View their personal profile

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

## Installation

Clone the repository:

git clone https://github.com/zianecelina-gif/sport-events-web-app.git

Install the dependencies:

pip install -r requirements.txt

## Run the Application

python controller.py

Then open in your browser:

http://127.0.0.1:5000

## Project Structure

controller.py → Flask routes and application logic
model.py → Database models and queries
static/ → CSS, JavaScript, images
view/ → HTML templates

## Goal of the Project

The goal of this project is to build a dynamic web application that connects people who want to practice sports and participate in local events.
